# WebAds.Studio ⚡

Premium, high-converting agency landing page designed to capture leads and showcase campaigns.

## 🚀 Features

*   **7-Day Deployment Card**: Modern grid statistics display.
*   **3D Interactive Tilts**: Hover animations on process stages.
*   **Drifting Cyberglows**: Slow-drifting gradient background spheres.
*   **Leads Admin Console**: LocalStorage database dashboard with CSV export (Password: `admin`).
*   **Vite Architecture**: Fast static compilation and bundling.

## 🛠️ Local Development

To run the project locally:

```bash
# Install dependencies
npm install

# Start local server on port 4000
npm run dev
```
